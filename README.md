## Website Goals & Objectives
- Showcase high-quality design work to attract new clients  
- Provide clear information about services and process  
- Enable easy project enquiries and contact with client-side validation  
- Deliver a modern, futuristic, and professional online presence  
- Implement SEO best practices for search visibility  

## Sitemap
- Home `index.html`  
- About Us `about.html`  
- Services `services.html`  
- Portfolio `portfolio.html`  
- Start a Project `enquiry.html`  
- Contact `contact.html`  

## Design Features (Part 2)
- External CSS stylesheet `css/style.css` for all styling
- Consistent black, gold `#D4AF77`, and white colour scheme  
- Responsive layout using Flexbox for mobile-friendly design  
- Subtle background animation using studio logo  
- Optimised, smaller portfolio images for performance  
- Gold/Black “colored box” inputs on Contact + Enquiry forms for luxury UI  

## Interactivity & Technical Features (Part 3)
- **JavaScript Form Validation**: `js/script.js` checks for empty fields on Contact + Enquiry forms
- **Error Handling**: Displays alert if any required field is missing
- **Processed Response**: Success message appears inside the form box after valid submission. No page reload.
- **SEO Files**: 
    - `robots.txt` - Allows search engines to crawl all pages
    - `sitemap.xml` - Lists all 6 pages for indexing
- **Semantic HTML**: Proper use of `<header>`, `<main>`, `<nav>`, `<footer>`, `<form>`, `<label>`
